# 📊 Acompanhamento Inbound

Sistema de acompanhamento de order em aberto, com status de RECDOCK. Acompanhamento de leed time do processo. 

## Sobreo o projeto

Este projeto tem como objetivo acomanhar o leed time de dias em que o status do pedido está como RECDOCK onde por sua vez não está alocado na posição correta. 

## 🧠 Funcionalidades

- 📥 Importação de dados (CSV, Excel, etc.)
- 🧹 Tratamento e limpeza de dados
- 📅 Padronização de datas
- 📊 Geração de métricas e indicadores
- 📈 Análise de performance
- 🔎 Identificação de inconsistências

## 🛠️ Tecnologias utilizadas

- Python
- Pandas
- Numpy
- Jupyter Notebook / Scripts Python
- Streamlit
